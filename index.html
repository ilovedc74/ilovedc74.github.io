RESEARCH PROJECT

Psychology Research Projects

Collaborative learning and Social Memory: Retrieval-Induced Forgetting

Psychological research project using the "Socially-Shared Retrieval-Induced Forgetting" paradigm to understand how social learning and practice impacts and magnifies memory performance. I created an experiement website with a JavaScript library jsPsych, and PHP to collect behavioural data. Directed by Dr Tsung-Ren Huang. I was responsible for data collection, data analysis and assisted in research design.

Tools used: R | Python (Spyder) | Excel | HTML, CSS | JavaScript | PHP


Caffeine and its Effect on Dynamic Visual Attention

To find out how caffeine would affect human visual attention, I measured covert attention with an eye-tracker during reading and manipulated caffeine intake. The statistical analysis was done with ANOVA. I generated random stimulus sentence list using Python and analysed my data with R. I discovered that caffeine accelerates attention orienting while information is being processed. Supervised by Dr Karina J Linnell.

Tools used: R | Python (Jupyter Notebook) | Eyelink 1000 (Eye-tracking hardware)
